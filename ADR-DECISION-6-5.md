### ADR-6.5: Recreate Views to be Classs Based

**Author:** Isaac Jessen

**Status:** Accepted | superseeds ADR 1.5

**AI Usage:**  
copilot used to generate base views that can be copied and altered to create the rest of the views.

**Context:**  
The views were not class based and thus did not follow the rubric

**Alternatives considered:**  

- Recreate all views

- Keep current views and accept loss of marks


**Decision:**  
Recreate all views to be class based and align with the rubric - still only creating base views for navigation and posting them for peer review
**Code reference:**  
views branch: group11_app/views.py 

**Consequences:**  
Views could be created incorrectly and then must be rewritten
